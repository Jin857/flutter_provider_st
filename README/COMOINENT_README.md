# 自定义组件

自定义组件的方法有三种:
1. 多个Widget组合
2. 通过CustomPaint绘制
3. 通过RenderObject绘制

# 多个Widget组合 
  这个比较简单，主要是依靠flutter提供的Widget组件进行封装，已实现需求逻辑。
  诸多例子：lib/component 文件下的组件可以学习。
